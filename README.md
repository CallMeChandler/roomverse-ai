# RoomVerse AI

RoomVerse AI is a multimodal room-understanding project that transforms room photos into structured scene representations and later into interior redesigns, game maps, and 3D previews.

## Current Progress
- Phase 1: Image loading and preprocessing pipeline
- Phase 2: MiDaS depth estimation pipeline
- Depth map visualization working

## Backend Stack
- Python
- PyTorch
- Pillow
- NumPy
- OpenCV
- Matplotlib

## Current Pipeline
Room image -> preprocessing -> MiDaS -> dense depth map

## Next Steps
- Phase 3: segmentation
- Phase 4: scene reasoning
- Phase 5: interior redesign
- Phase 6: game map generation
- Phase 7: 3D preview
